Your1 one-stop Minecraft modding shop!


Modding software

Here you can find a collection of tools you can use to mod Minecraft on console.

Icon

PCK Studio

This program can be utilized to create and modify .PCK files, which are used in skin packs, texture packs, and mashup packs.

Icon

ARC Studio

This program can be used to modify existing .ARC files, which are used in the display of strings, rendering of GUIs, and the loading of structure files.

Icon

FUI Studio

This program can be used to create and modify .FUI files, which are scenes containing UI objects, their positions, and textures.

Icon

Texture Builder Tool

This program can be used to create the structure used for minecraft console texture packs!

Icon

Spark Model Editor

This program can be used to create and edit the models.bin file used to customize entity models in texture packs!

Development

Throughout my years as a Modder for Minecraft on the legacy consoles, I've learned much about the game itself as well as the consoles it runs on, because of this, combined with the fact that many creators have stopped modding either on legacy consoles, or completely, I have made it my mission to make tools that are both open source, and easy to use, to allow anyone, no matter how new, to dive in head first and make incredible mods.
